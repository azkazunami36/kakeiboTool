# 家計簿アプリ
Excelでとりあえず金額管理をしていましたが、ふと作りたくなりました。家計簿アプリで友人の分を代替で購入したということを記す機能なんてないっぽいので、自作しました。

Node.jsで実行するものです。環境にNode.jsさえ入っていれば大体実行できます。テスト済みの環境はmacOSですが、きっとLinuxでも使えます。Windowsは分かりません。

# 実行方法
```sh
npm start
```
たったこれだけで`http://localhost:3000`が立ち上がります。一応リポジトリを公開するにあたって、立ち上がったことを伝える旨の「起動したよー」とだけ表示するようにしています。
